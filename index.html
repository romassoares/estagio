Transparência

Esta página foi criada com o intuito de disponibilizar informações relevantes para a sociedade, sendo de fácil compreensão e acesso para a sociedade interessada.

Alguns dos sites utilizados:

  1. http://sistemas.tce.pi.gov.br/api/portaldacidadania/prefeituras
  2. http://sistemas.tce.pi.gov.br/api/portaldacidadania/despesas/
  3. https://servicodados.ibge.gov.br/api/v1/localidades/microrregioes/22005/distritos

Como fazer:

  1. criar um arquivo HTML;
  2. carregar um modelo de mapa:
    3. https://leafletjs.com/reference-1.7.1.html#map-option
  3. escolher região para ser carregada inicialmente;
  4. marcar pontos ou delimitar regiões com coordenadas geográficas;
  5. criar funções com uma linguagem de programação para “pegar” as informações das API’s;
  6. carregar as informações das funções no mapa.
